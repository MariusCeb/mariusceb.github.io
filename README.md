# Portfolio — Ceban Marius

Personal portfolio showcasing web design and development projects.

## Tech

Static HTML/CSS/JS — no framework, no build step. IBM Plex Mono typeface via Google Fonts.

## Structure

```

├── index.html          Homepage
├── about.html          About page
├── projects.html       Projects grid
├── donkey.html         Project 01 — Donkey Joe
├── pizzeria.html       Project 02 — Uà Art&Pizza
├── serenamente.html    Project 03 — Serenamente
├── SpaceBlog.html      Project 04 — Space Blog
├── unishare.html       Project 05 — UniShare
├── curriculumMarius.pdf  CV download
└── foto/

    └── unishare/       UniShare screenshots
```

## Features

- Light/dark theme with localStorage persistence
- Custom cursor
- Page transition animations
- Responsive design (mobile + desktop)
- Lightbox image viewer
- Scroll-reveal animations
